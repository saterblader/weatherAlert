# weatherAlert
 A script to querey NOAA for weather alerts for a given NOAA County Code as found at: 
 https://alerts.weather.gov/
 Requires tcl packages http 2 & tls >=1.7
 
 Usage:
 ./weatherAlert -zone \<NOAA County Zone Code\> [-quiet yes]
